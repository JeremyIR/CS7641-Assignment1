#### Steps to Run Code

1) Clone github repository: https://github.com/JeremyIR/CS7641-Assignment1
2) Navigate to directory of repository
3) Make sure you have python3 and pip3 installed on your computer
4) Run command "pip3 install -r requirements.txt"
5) Preprocess data by running "python3 processdata.py"
6) To run code for decision tree run "python3 decisionTree.py"
7) To run code for neural networks run "python3 neural.py"
8) To run code for boosting run "python3 boostedDT.py"
9) To run code for Support Vector Machines run "python3 svm.py"
10) To run code for K-Nearest Neighbors run "python3 knn.py"
